## Pre-generate the two values the script asks for

**Manual will prompt during ./arculus-setup.sh for:**

- a MySQL password (backend uses it), and 
- a 256-bit encryption key for JWT signing.

**Terraform can**: Generate those two values up front and print them as outputs (or store them in SSM), so students paste them when the script asks. You are not bypassing the prompt; youâ€™re just avoiding weak, ad-hoc secrets

**TF Sketch**
```
resource "random_password" "mysql" { length=16 special=false }
resource "random_id" "jwt"        { byte_length=32 } # 256-bit

output "mysql_password" { value = random_password.mysql.result  sensitive = true }
output "jwt_256_key"    { value = random_id.jwt.b64_std         sensitive = true }

```
